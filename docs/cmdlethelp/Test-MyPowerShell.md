# Test-MyPowerShell

## SYNOPSIS
Test my PowerShell.

## SYNTAX

```
Test-MyPowerShell [<CommonParameters>]
```

## DESCRIPTION
Test my current PowerShell session to know whether the version is up-to-date or not.

## EXAMPLES

### EXAMPLE 1
```
Test-MyPowerShell
```

Return information about the version of my current PowerShell session.
